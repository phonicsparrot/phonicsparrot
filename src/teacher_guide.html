Teacher Setup Guide

Complete guide to running and managing Phonics Parrot in your classroom.

🚀 Quick Start

  • 1
    Extract the filesExtract the downloaded .7z or .zip file onto your desktop or computer folder.
  • 2
    Start the local serverDouble-click the START.bat script file in the extracted folder.
  • 3
    Open Phonics ParrotA browser window will automatically launch to http://localhost:3000.
  • 4
    Keep the server runningMinimise the black console terminal window that popped up. It must remain open while students are playing.
💡
Manual URL: If the web browser doesn't open automatically, just launch Chrome or Microsoft Edge and navigate directly to http://localhost:3000.
📡
Classroom Display & Tablets: Other devices on the same Wi-Fi network (such as tablets or smart screens) can access the app at the network address printed in the terminal (e.g., http://10.x.x.x:3000).

☁️ Google Drive & Sheets Integration

Connecting Phonics Parrot to Google Drive enables student voice recordings to upload directly into your cloud folders, and automatically logs performance metrics to a Google Sheet.

Step 1: Set up Google Apps Script

  • 1
    Open Apps ScriptNavigate to your browser and open Google Apps Script, then click New Project.
  • 2
    Clean the fileDelete the default starter code inside the script window.
  • 3
    Copy & modifyCopy the AppScript code provided on the right using the copy button, then paste it in.
  • 4
    Link SpreadsheetReplace the value of SPREADSHEET_URL at the very top of the script with your real Google Sheet URL, then save with Ctrl+S.

Step 2: Deploy as a Web App

  • 5
    Initiate DeploymentClick the blue Deploy button in the top right corner, then select New deployment.
  • 6
    Configure Web AppClick the gear icon (⚙️) next to "Select type", choose Web app, and set Execute as: Me & Who has access: Anyone.
  • 7
    Copy Script URLClick Deploy, authorize permissions, and copy the generated Web App URL.

Step 3: Connect to Phonics Parrot

  • 8
    Edit config fileOpen drive_config.json in your app folder using Notepad.
  • 9
    Paste URLReplace the placeholder URL inside with your Web App URL, save the file, and restart `START.bat`.
appsscript/Code.gs 
var SPREADSHEET_URL = "https://docs.google.com/spreadsheets/d/YOUR_SPREADSHEET_ID_HERE/edit";
var RECORDINGS_FOLDER_NAME = "Phonics Parrot Recordings";

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var driveUrl = "";

    if (data.audioData) {
      driveUrl = saveAudioToDrive(data);
    }

    logToSheet(data, driveUrl);

    return ContentService
      .createTextOutput(JSON.stringify({ result: "success", driveUrl: driveUrl }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ result: "error", message: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function saveAudioToDrive(data) {
  var audioBytes = Utilities.base64Decode(data.audioData);
  var filename = data.filename || ((data.student || "guest") + "_recording.webm");
  var blob = Utilities.newBlob(audioBytes, "audio/webm", filename);

  var rootFolders = DriveApp.getFoldersByName(RECORDINGS_FOLDER_NAME);
  var rootFolder = rootFolders.hasNext() ? rootFolders.next() : DriveApp.createFolder(RECORDINGS_FOLDER_NAME);

  var className = (data.class || "Guest").replace(/[\/\\]/g, "_");
  var studentName = (data.student || "Guest").replace(/[\/\\]/g, "_");
  var activity = (data.activity || "activity").replace(/[\/\\]/g, "_");

  var classFolder = getOrCreateSubfolder(rootFolder, className);
  var studentFolder = getOrCreateSubfolder(classFolder, studentName);
  var actFolder = getOrCreateSubfolder(studentFolder, activity);

  var file = actFolder.createFile(blob);
  file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);
  return file.getUrl();
}

function getOrCreateSubfolder(parent, name) {
  var iter = parent.getFoldersByName(name);
  return iter.hasNext() ? iter.next() : parent.createFolder(name);
}

function logToSheet(data, driveUrl) {
  var ss = SpreadsheetApp.openByUrl(SPREADSHEET_URL);
  var student = (data.student || "Guest").trim();

  var sheet = ss.getSheetByName(student);
  if (!sheet) {
    sheet = ss.insertSheet(student);
    sheet.appendRow([
      "Timestamp", "Class", "Activity",
      "Target Word / Phrase", "Phoneme",
      "Spoken Word", "Score", "Verdict",
      "Audio Recording Link"
    ]);
    sheet.getRange(1, 1, 1, 9).setFontWeight("bold");
    sheet.setFrozenRows(1);
  }

  sheet.appendRow([
    data.timestamp || new Date().toISOString(),
    data.teacher || data.class || "",
    data.activity || "",
    data.target || "",
    data.phoneme || "",
    data.spoken || "",
    data.score !== undefined ? data.score : "",
    data.verdict || "",
    driveUrl || ""
  ]);
}

💾 Local Storage & Safety

Even if you choose not to set up the Google Cloud Integration, your student records are fully saved locally:

  • Audio recordings are stored directly in the recordings/ folder.
  • Grades and progress logs are written to performance_log.json.

This means you can always view student work locally, even without an active internet connection.

❓ Frequently Asked Questions

Does the application require internet?
Yes, but only for the speech-to-text microphone capability (which queries the browser's speech recognition engine). The interactive lessons and data storage are fully local and offline.
Can multiple tablets connect to a single host computer?
Yes. Simply open the URL displayed in the server terminal (e.g. http://192.168.x.x:3000) on all tablets connected to the same Wi-Fi.
What happens if a port already in use error occurs?
Phonics Parrot will automatically detect the port conflict and bind to the next available port (e.g. 3001). Check the terminal output to verify the port number.
Which web browsers are supported?
We recommend using Google Chrome or Microsoft Edge, as Firefox has limited Speech Recognition support.
